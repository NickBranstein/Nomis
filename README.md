# Nomis
A new take on an endless classic...plus mechs!  Nomis is a HTML5 game created for the 2015 js1k3games competition.

## Special Thanks
Nomis was created by Nick Branstein, David Bjarnson, and Justin Tindle

## Running
```
npm install
gulp
```
